Curso de React JS

O que é React?

  • Biblioteca JavaScript para desenvolvimento front-end
  • Categoria SPA (Single Page Application)
  • Possui a arquitetura baseada em componentes
  • Mantido pelo Facebook/Meta
  • É open source

Pré-requisitos do React

  • Precisa do Node.js
  • Gerenciador de pacotes npm para iniciar projetos

Hello World em React

  • Utilizaremos o Vite para criar aplicações
  • Digitar no terminal: npm create vite@latest e seguir as opções

Estrutura do React

    Pastas e arquivos muito importantes:

  • node_modules: dependências do projeto
  • public: assets e arquivos estáticos
  • src: código da aplicação
  • src/index.js: arquivo de inicialização da aplicação
  • src/App.js: componente principal inicial - pode ser modificado
