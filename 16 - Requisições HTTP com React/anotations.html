Requisições HTTP com React

JSON Server

  • Pacote npm
  • Simula uma API, podendo fazer requisições GTTP
  • Não precisa de um back-end

A importância do useEffect

  • Faz com que seja possível controlar a execução de uma ação
  • Recursos podem ser re-executados a cada re-renderização
  • O componente é re-renderizado a cada mudança
  • Possui um array de dependências que coordena o que permite a execução do código
  • Muito comum nas requisições HTTP

Resgatando dados com React

  • Usar useState para salvar dados
  • Utilizar useEffect para chamar a API apenas quando necessário
  • Realizar a requisição da API com alguma ferramente, como Axios ou Fetch API
